# J11 — La vie associative : toutes les assos de Mézières

---

🤝 **Sport, culture, loisirs, entraide… La vie associative de Mézières est riche. Découvrez-la sur MAT !**

La commune compte de nombreuses associations actives. MAT vous permet de toutes les retrouver facilement, en un seul endroit.

**Ce que vous trouvez dans l’annuaire des associations :**
- 🏃 Les associations sportives
- 🎭 Les associations culturelles
- 👨‍👩‍👧 Les associations familiales et d’entraide
- 📞 Les contacts pour les rejoindre ou les soutenir

**Pourquoi c’est utile ?**
Vous venez d’arriver dans la commune ? Vous cherchez une activité pour vous ou vos enfants ? Vous souhaitez vous investir dans la vie locale ? L’annuaire de MAT est votre point de départ idéal.

**Comment y accéder :**
Ouvrez MAT → Rubrique **Associations**.

👉 **La vie de quartier commence ici — rejoignez une association de Mézières !**
