# J04 — Les notifications push : restez informé sans lever le petit doigt

---

🔔 **MAT peut vous prévenir directement sur votre téléphone — même quand l’application est fermée !**

Pas besoin d’ouvrir MAT pour rester informé. Activez les notifications et laissez l’appli travailler pour vous.

**Ce que vous pouvez recevoir :**
- 🗑️ Rappel de collecte des poubelles (la veille au soir, au moment de les sortir)
- 📰 Les nouvelles actualités publiées par la mairie
- 📅 Les prochains événements importants
- ⚠️ Les alertes urgentes de la commune

**Comment activer les notifications ?**
1. Ouvrez MAT
2. Allez dans les paramètres (⚙️)
3. Activez les notifications souhaitées
4. Acceptez l’autorisation quand votre téléphone vous le demande

C’est tout ! Vous pouvez choisir exactement ce que vous souhaitez recevoir.

**Note :** Les notifications fonctionnent uniquement si vous avez installé MAT sur votre écran d’accueil (voir J01).

👉 **L’information vient à vous — vous ne cherchez plus, on vous prévient !**
